# Photos & Videos

The `photos/` directory is excluded from the repository (see `.gitignore`).
All 32 files are listed in `photos.json` with their GPS coordinates and type.

To restore the `photos/` directory, re-export the following files from your
iPhone/iCloud Photos library into a `photos/` folder in the project root.

After re-exporting, regenerate the thumbnails by running:

```sh
./make-thumbnails.sh
```
